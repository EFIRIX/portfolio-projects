Sanitized For Review

This repository is intentionally a portfolio snapshot, not a production backup.

  • Local `.env` files and production secrets are excluded.
  • Dependencies, build outputs, binaries, logs, trained models, and heavy generated documents are excluded.
  • Private data and runtime datasets are excluded from ML and automation projects.
  • Some projects are case-study snapshots because they require private services or local data to run fully.
