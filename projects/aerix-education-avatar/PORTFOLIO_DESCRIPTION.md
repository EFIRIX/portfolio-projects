# AERIX Education Avatar

AERIX Education Avatar is a real-time AI education MVP built around a TypeScript client/server architecture. It explores session streaming, avatar animation payloads, audio capture/playback, latency budgets, and policy boundaries for an interactive learning assistant.

## Portfolio Value

- Demonstrates real-time product thinking: streaming, audio, state, and latency constraints.
- Includes architecture and API documentation, not only source files.
- Shows how an AI-facing frontend and backend can be structured for an education use case.
