# Second Brain Knowledge System

Second Brain Knowledge System - визуальный портфолио-кейс персональной базы знаний на markdown/Obsidian: файловая структура, граф связей и системная организация заметок.

## Portfolio Value

- Демонстрирует системное мышление и knowledge management.
- Показывает работу с Obsidian/markdown-хранилищем и графом связей.
- Дополняет техническое портфолио проектом про организацию знаний и личную продуктивность.
