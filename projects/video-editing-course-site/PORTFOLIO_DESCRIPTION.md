# Video Editing Course Site

Video Editing Course Site is an older static website for an educational project about video editing. It includes HTML, CSS, JavaScript, and SVG/image assets.

## Portfolio Value

- Demonstrates static web publishing around an educational/course topic.
- Useful as archive material showing earlier frontend and presentation work.

## Public Snapshot

Large media files, videos, audio, PDFs, and local state were excluded. The project was staged through `Downloads` before being copied into this repository.
