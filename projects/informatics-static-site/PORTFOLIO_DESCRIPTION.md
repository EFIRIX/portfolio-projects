# Informatics Static Site

Informatics Static Site is a small educational static website. The original folder included media-heavy assets, but the portfolio version keeps a lightweight source snapshot.

## Portfolio Value

- Demonstrates basic static site structure for educational content.
- Included as an archive project showing earlier web work.

## Public Snapshot

Large media, audio, video, image-heavy assets, and PDFs were excluded where appropriate. The public version is meant as a lightweight code review snapshot.
