# EGE Materials Generator

EGE Materials Generator is a Python tooling project for creating educational exam-preparation materials, including scripts for transforming source content into printable resources.

## Portfolio Value

- Demonstrates practical education tooling and document-generation logic.
- Shows automation applied to a real learning/preparation workflow.
- Useful as a niche but concrete example of Python productivity tooling.

## Public Snapshot

Generated PDFs, DOCX files, previews, and text caches were excluded. The public version keeps the Python generation scripts only.
