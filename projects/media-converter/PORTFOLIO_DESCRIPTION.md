# Media Converter

Media Converter is a Python desktop utility for converting media files. The project includes GUI entry points, a command entry point, installation script, packaging metadata, and dependency definitions.

## Portfolio Value

- Demonstrates desktop utility development in Python.
- Shows packaging-oriented structure, not just a script.
- Useful as a compact portfolio project for practical tooling.

## Public Snapshot

DMG, build, dist, package, and binary icon artifacts were excluded. The repository keeps the application source and dependency list.
