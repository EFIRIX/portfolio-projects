# Media Converter - Liquid Glass Edition

**Красивое приложение для конвертации медиафайлов** с интерфейсом в стиле Liquid Glass.

## 🎨 Дизайн

- **Liquid Glass Style** - Прозрачные элементы с эффектом стекла и размытием
- **Анимации** - Плавное появление окна, свечение кнопок
- **Современный интерфейс** - Чистый минималистичный дизайн
- **Перетаскивание файлов** - Drag & Drop поддержка
- **Безрамочное окно** - Перетаскивание за любую область

## 📁 Поддерживаемые форматы

### 🎬 Видео
`.mp4`, `.mkv`, `.avi`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`

### 🎵 Аудио
`.mp3`, `.wav`, `.aac`, `.ogg`, `.flac`, `.m4a`

### 🖼️ Изображения
`.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`

## ⚡ Возможности

- **Выбор файла** - через диалог или перетаскиванием (Drag & Drop)
- **Автоматический выбор формата** на основе типа файла
- **Прогресс-бар** с анимацией
- **Поддержка FFmpeg** для видео и аудио
- **Поддержка ImageMagick** для изображений
- **Безопасность** - проверка перед конвертацией
- **Самодостаточное приложение** - все зависимости внутри .app

## 📥 Установка

### 🍎 Через DMG (Рекомендуется)

1. Откройте файл `MediaConverter.dmg`
2. Перетащите `Media Converter.app` в папку **Приложения**
3. Запустите из Launchpad или папки Приложения

### 🐍 Из исходников

```bash
# Установите зависимости
brew install ffmpeg imagemagick
pip install PySide6

# Запуск
python3 app.py
```

## 🚀 Использование

1. **Выберите файл** - Перетащите файл в область или нажмите для выбора через диалог
2. **Выберите формат** - Укажите целевой формат из выпадающего списка
3. **Нажмите "Конвертировать"** - Запустите процесс конвертации
4. **Готово!** - Новый файл появится в той же папке, что и исходный

## 📦 Сборка из исходников

```bash
# Установите зависимости
pip install PySide6 pyinstaller

# Соберите .app
pyinstaller --windowed --name "Media Converter" --icon=icon.icns app.py

# Скопируйте бинарники ffmpeg и convert в Contents/MacOS/
cp /opt/homebrew/bin/ffmpeg dist/Media\ Converter.app/Contents/MacOS/
cp /opt/homebrew/bin/convert dist/Media\ Converter.app/Contents/MacOS/

# Создайте DMG
# (используйте hdiutil или create-dmg)
```

## 🎯 Технологии

- **Python 3.9+**
- **PySide6 (Qt for Python)** - Для красивого GUI
- **FFmpeg** - Конвертация видео и аудио
- **ImageMagick** - Конвертация изображений

## 📝 Примечания

- Приложение автоматически находит `ffmpeg` и `convert` в стандартных путях
- В собранном .app бинарники встроеныInside
- Для работы с большими файлами требуется достаточно места на диске
- Конвертация может занять время в зависимости от размера файла

## 🔧 Устранение неполадок

Если приложение не запускается:
1. Убедитесь, что у вас установлен **macOS 11.0+** (Big Sur или новее)
2. Проверьте, что в папке `Contents/MacOS/` есть файлы `ffmpeg` и `convert`
3. Попробуйте запустить из терминала:
   ```bash
   /Applications/Media\ Converter.app/Contents/MacOS/Media\ Converter
   ```
