# LifeOS

LifeOS is a Swift project for a personal operating system concept: a native-style app for life tracking, reports, dashboard views, strategy, statistics, and local data persistence.

## Portfolio Value

- Demonstrates Swift app structure with multiple view models and a local storage layer.
- Shows native product thinking rather than only web/backend work.
- Works well as a portfolio piece for mobile and personal productivity concepts.
