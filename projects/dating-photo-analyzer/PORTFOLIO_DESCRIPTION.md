# Dating Photo Analyzer

Dating Photo Analyzer is a small React/Vite browser demo with a polished UI for analyzing dating profile photos. It is included as a compact frontend experiment rather than a production AI system.

## Portfolio Value

- Demonstrates quick frontend prototyping and UI composition.
- Useful as a lightweight browser demo that can be viewed from the portfolio page.
- Shows ability to package a small idea into a usable interactive interface.

## Public Snapshot

Dependency folders were excluded. The source and build output needed for preview are included.
