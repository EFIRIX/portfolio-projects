# Telegram Automation Bot

Telegram Automation Bot is a Python automation project with a GUI flow, secure setup logic, storage layer, web module, packaging templates, and tests. It is designed around Telegram bot automation while keeping sensitive credentials out of source control.

## Portfolio Value

- Demonstrates automation engineering with real runtime concerns: config, storage, network modes, packaging, and tests.
- Shows secure configuration patterns and masking of sensitive values.
- Useful as a portfolio example for bot, desktop utility, and operational automation work.

## Public Snapshot

The public version excludes `.env`, virtual environments, logs, executable builds, and local runtime data. Test token fixtures were replaced with safe placeholder values.
