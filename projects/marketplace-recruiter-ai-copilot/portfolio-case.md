# Portfolio Case: Marketplace Recruiter AI Copilot

## Коротко

Собрал n8n-автоматизацию для первичного скоринга кандидатов на вакансии компаний, которые работают с Wildberries и Ozon. Workflow принимает вакансию и резюме через webhook, оценивает релевантность кандидата, генерирует summary для рекрутера, первое сообщение кандидату, вопросы для интервью и логирует примерный расход токенов.

## Задача

В рекрутинге много ручной первичной обработки: нужно быстро понять, есть ли у кандидата опыт с WB/Ozon, рекламой, аналитикой, карточками, остатками, MPStats/Excel и другими частыми требованиями. Цель проекта - превратить резюме и вакансию в понятную карточку кандидата за один webhook-запрос.

## Что реализовано

- n8n workflow для импорта.
- Webhook input под вакансию и кандидата.
- Mock AI scoring без API-ключей, чтобы кейс можно было показать сразу.
- Fit score от 0 до 100 и verdict: `strong_match`, `needs_review`, `low_match`.
- Автоматическое summary для рекрутера.
- Персональное outreach-сообщение кандидату.
- Follow-up questions для первичного интервью.
- Google Sheets append с готовой структурой колонок.
- Оценка примерного расхода токенов и стоимости.

## Чем полезно бизнесу

- Рекрутер быстрее видит, кого брать в работу.
- Кандидаты получают более персональные сообщения.
- Вся первичная оценка сохраняется в таблицу.
- AI-логика отделена от интеграций: mock node можно заменить на Claude/OpenAI без перестройки workflow.

## Что бы добавил в production

- Реальный Claude или OpenAI node с JSON-schema output.
- Telegram alert для сильных кандидатов.
- Отдельную таблицу для логов ошибок и стоимости.
- Ручную корректировку скоринга рекрутером для последующей калибровки промпта.

## Формулировка для отклика

Последним для себя я плотно открыл связку Codex + n8n: использую Codex как инженерного напарника для быстрой сборки структуры проекта, а n8n - как слой автоматизации, где можно быстро превращать идею в рабочий бизнес-процесс. Для этой вакансии собрал мини-кейс Marketplace Recruiter AI Copilot: автоматизацию, которая скорит кандидатов под WB/Ozon-вакансии, пишет summary рекрутеру, сообщение кандидату и сохраняет результат в Google Sheets.
