# Marketplace Recruiter AI Copilot

Marketplace Recruiter AI Copilot is an n8n workflow and automation case for recruiting operations. It structures incoming candidate/job data, estimates AI-processing cost, prepares recruiter-facing summaries, and routes results into an operational workflow.

## Portfolio Value

- Demonstrates automation design with a real business workflow.
- Shows no-code/low-code integration thinking through n8n.
- Useful as a portfolio case for AI operations and marketplace/recruiting automation.

## Public Snapshot

Local data files and runtime state were removed. The public version keeps the workflow JSON, README, and case notes.
