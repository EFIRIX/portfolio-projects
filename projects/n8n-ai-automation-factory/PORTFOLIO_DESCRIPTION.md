# n8n AI Automation Factory

n8n AI Automation Factory - портфолио-проект с несколькими automation workflow: большая Telegram AI content factory, micro auto-debug workflow, маленький HTTP prototype и marketplace recruiter copilot.

## Portfolio Value

- Показывает умение проектировать автоматизации не только как простые цепочки, но и как полноценные workflow-системы с ветвлением, fallback, модерацией и логированием.
- Демонстрирует работу с Telegram, HTTP requests, AI JSON parsing, callback actions, Google Sheets/операционными выходами и n8n static data patterns.
- Хорошо дополняет портфолио как отдельный блок no-code/low-code automation engineering.

## Public Snapshot

Workflow JSON очищены перед публикацией: credentials, webhook IDs, pin/static data, token-like строки и персональные IDs удалены или заменены placeholder-ами. Скриншоты добавлены только как визуальное превью интерфейса n8n.
