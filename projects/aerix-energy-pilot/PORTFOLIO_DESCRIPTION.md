# AERIX Energy Pilot

AERIX Energy Pilot is an energy intelligence prototype focused on forecasting, anomaly detection, optimization, AutoML, explainability, monitoring, and operational scenario analysis.

## Portfolio Value

- Demonstrates applied ML and analytics work beyond a basic notebook.
- Includes modular Python source, tests, configuration, forecasting logic, and monitoring components.
- Strong portfolio fit for AI, industrial analytics, and decision-support systems.

## Public Snapshot

Datasets, trained model files, logs, generated plots, and runtime outputs were removed from the public version. The repository keeps the source, tests, requirements, and documentation needed for review.
