# Слайд 1. Заголовок
- AERIX Energy Pilot
- ИИ-платформа управления энергопотреблением

# Слайд 2. Проблема
- Пиковые нагрузки
- Рост стоимости генерации
- Выбросы CO2
- Отсутствие предиктивного управления

# Слайд 3. Цель проекта
- Создание ИИ-системы оптимизации энергопотребления

# Слайд 4. Решение
- Прогноз нагрузки + выявление пиков + оптимизация

# Слайд 5. Архитектура
- Data → AI → Peak Detection → Optimization → Экономика и CO2

# Слайд 6. Данные
- 121 296 часов реальной энергосети (PJM)

# Слайд 7. Модели
- Linear Regression
- Random Forest

# Слайд 8. Качество модели
- MAE: 1334 МВт
- Улучшение: 3.71%

# Слайд 9. Пиковый анализ
- Выявлены реальные вечерние пики

# Слайд 10. Эффект оптимизации
- Снижение пика: 16.9%
- Снижение мощности: 10 169 МВт

# Слайд 11. Экономика
- ≈ 76 млн ₽ экономии за пиковый час

# Слайд 12. Экология
- ≈ 4.6 тыс. тонн CO2 снижения

# Слайд 13. Применение в РФ
- Энергосистемы
- Промышленные предприятия
- Металлургия
- Нефтехимия
- Дата-центры

# Слайд 14. AERIX как платформа
- Переход от модуля энергии к экосистеме управления инфраструктурой

# Слайд 15. Roadmap
- MVP → Пилот на предприятии → Интеграция → Экосистема

# Слайд 16. Финал
